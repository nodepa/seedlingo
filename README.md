Seedlingo is a modern literacy tool designed specifically for adults who are already masters of their native language but haven't yet had the opportunity to master the written word.

While many apps focus on teaching new languages, Seedlingo focuses on empowerment. The inability to read can be a barrier to full participation in society. Seedlingo acts as a bridge, taking the rich vocabulary you already possess and turns it into a tool for confidence, independence, and a connected life.

## ✨ Why Seedlingo?

*Cultivating the words you speak into the world you read.*

Seedlingo uses a clean, distraction-free interface to help users map the sounds and meanings they already know to their written forms. By using optimized review cycles, the app ensures that recognition becomes second nature, allowing learners to focus on understanding and engagement.

- **A fertile foundation:** Leverage your existing spoken fluency. You already know the words; we simply help you recognize their written forms.
- **Respect the roots:** Designed specifically for adult learners. Just clear, practical progress toward reading.
- **Sow & reap:** Using a focused, distraction-free interface, the app helps you plant words in your long-term memory through optimized recognition cycles.
- **Green pastures:** Literacy makes life easier. From reading official documents to navigating your community, Seedlingo helps you engage fully in a text-driven world.

---

Seedlingo was originally developed as [立爱种字 LiAiZhongzi.com](https://liaizhongzi.com) from ideas by [立爱教育 Li Ai Education](https://liaieducation.com), to help Putonghua speaking adults learn to read simplified Chinese characters.

To try the Android app, [download version 1.5.2](https://github.com/nodepa/seedlingo/releases/download/v1.5.2/seedlingo_v1.5.2_64.apk) or check the [release history](https://github.com/nodepa/seedlingo/releases) for a more recent version. Download to your phone, then install. You may have to enable the setting "Install from unknown sources" before your phone will allow the installation.

There is also an English language version available for demonstration purposes [online at en.seedlingo.app](https://en.seedlingo.app) or [downloadable for Android](https://github.com/nodepa/seedlingo-en/releases).

## 🚀 Quick start

Help us grow this mission by setting up a local development environment:

1. **Clone the repository**

    ```bash
    git clone https://github.com/nodepa/seedlingo.git
    ```

2. **Install dependencies**

    ```bash
    cd seedlingo
    npm install
    ```

3. **Launch the app**

    ```bash
    # preview mode
    npm start    # > http://localhost:4173 in your browser

    # developer mode
    npm run dev  # > http://localhost:5173 in your browser

    # run tests
    npm test
    ```

### Prerequisites

- Install [git](https://git-scm.com) if not present.
- Install [Node.js](https://nodejs.org), which includes
  [npm](https://www.npmjs.com/get-npm) (Node Package Manager).

  - If you *do not* use [Volta](https://volta.sh/),
    **use the version specified in the `volta`-section of `package.json`**
    when installing [Node.js](https://nodejs.org) according to your preference.

  - If you *do* use [Volta](https://volta.sh/),
    the `package.json`-specified Node version
    will automatically be installed and used
    when commands are executed.



### Content

Play around with your own content by replacing the `content/` folder (parallel to the `app/` folder) with your own content.  See the [Seedlingo documentation](https://seedlingo.com/content/content.html) about formats.
